# Manuel d'Utilisateur - Matos

## Accès à l'Application
> **Note importante :** Vos données sont stockées localement dans votre navigateur (**LocalStorage**). Elles ne sont pas envoyées sur un serveur, ce qui garantit votre confidentialité.

## Navigation Principale
L'en-tête (Header) contient trois onglets principaux :
- **Tableau de bord** : Vue d'ensemble des statistiques et liste du stock.
- **Nouveau** : Formulaire pour ajouter une nouvelle entrée de matériel (vue par défaut).
- **Paramètres** : Gestion du catalogue de catégories et de produits.

## Gestion du Stock
### Ajouter un matériel
1. Cliquez sur l'onglet **"Nouveau"**.
2. **Catégorie** (Optionnel) : Sélectionnez une catégorie (ex: Extincteurs EAU).
3. **Nom du produit** : Choisissez un produit dans la liste déroulante (qui s'adapte à la catégorie) ou saisissez-le manuellement.
4. **Quantité** : Indiquez le nombre d'unités.
5. **Client** (Optionnel) : Saisissez le nom du client ou de l'agence.
6. **Statut** : Choisissez entre *À récupérer*, *Validé* ou *Stand By*.
7. **Dates** : Vous pouvez renseigner une date prévue de récupération ou une date de réception.
8. Cliquez sur **"Créer le matériel"**.

### Consulter et Filtrer le stock
Dans le **Tableau de bord**, vous disposez de plusieurs outils :
- **Barre de recherche** : Tapez le nom d'un produit ou d'un client pour filtrer instantanément.
- **Filtres rapides** : Cliquez sur les cartes de compteur en haut (Total, À Récupérer, etc.) pour filtrer la liste par statut.
- **Listes déroulantes** : Filtrez par catégorie ou statut spécifique.

### Actions sur un matériel
Chaque carte de matériel dispose de boutons d'action rapide :
- **Icône Cocher (Vert)** : Valide le matériel (statut passe à "Validé").
- **Icône Horloge (Orange)** : Met le matériel en attente (statut passe à "Stand By").
- **Icône Crayon (Bleu)** : Ouvre le formulaire pour modifier les informations.
- **Icône Poubelle (Rouge)** : Supprime définitivement l'entrée (une confirmation sera demandée).

## Configuration (Paramètres)
Vous pouvez personnaliser votre catalogue de travail :
- **Catégories** : Ajoutez ou supprimez des catégories de matériel.
- **Produits** : Ajoutez des produits spécifiques en les liant à une catégorie pour qu'ils apparaissent automatiquement dans les suggestions du formulaire.

## Notifications
Des messages colorés s'affichent en bas à droite de l'écran pour confirmer chaque action (ajout, modification, suppression, changement de statut).
